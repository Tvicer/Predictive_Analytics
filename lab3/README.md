# Лабораторная работа №3. Классификация и прогнозирование метеоданных

## Команда для запуска
```bash
pip install -r requirements.txt
jupyter notebook Predictive_analytics_lab3.ipynb
```

## Результат
В ходе лабораторной работы была реализована комплексная двухэтапная система анализа временных рядов метеоданных, объединяющая исследование структуры данных, построение признаков, классификацию климатических режимов и прогнозирование температуры на среднесрочном горизонте. Особое внимание уделялось не только качеству моделей, но и интерпретации результатов через визуальный и статистический анализ ошибок.

Выполненный разведочный анализ позволил выявить выраженную сезонность температур, различия климатических профилей городов, характер распределения ключевых метеорологических параметров и их взаимные корреляции. Инженеринг признаков включал календарные и циклические компоненты, лаговые и скользящие статистики, динамические показатели и климатические нормы с аномалиями, что обеспечило информативное представление временных рядов для последующих моделей.

На задаче классификации городов и типов климата были опробованы как табличные модели, так и специализированные алгоритмы для временных рядов (MiniROCKET, DTW‑kNN), показавшие высокую точность и устойчивость к вариативности погодных условий. Для прогноза температуры реализованы несколько подходов — сезонный наивный, SARIMA по аномалиям, Prophet и ансамбль моделей LightGBM по расширенному признаковому пространству — и проведено их сравнение по метрикам MAE, RMSE, MAPE, WAPE и точности направления изменения.

Статистический анализ остатков (тесты Шапиро–Уилка, Льюнга–Бокса, Уайта), а также разбор ошибок по сезонам и горизонтам прогноза показали, в каких режимах модели работают надёжно, а где наблюдаются систематические отклонения, связанные с резкими погодными переходами или нестандартными аномалиями. На основе лучших комбинаций «город–модель» построен интегрированный пайплайн: по короткому окну истории он сначала идентифицирует климатический режим, затем автоматически подбирает оптимальную модель прогноза и формирует траекторию температуры на заданный период.

В совокупности результаты демонстрируют, что специализированные методы анализа временных рядов и грамотно спроектированные признаки позволяют эффективно решать задачи климатической классификации и среднесрочного прогнозирования погоды, а интеграция классификации и прогноза в единый пайплайн делает систему ближе к реальным прикладным сценариям использования.
