# Лабораторная работа №2. Виртуальный датчик для контроля процесса обжига в печи

## Команда для запуска
```bash
pip install -r requirements.txt
jupyter notebook Predictive_analytics_lab2.ipynb
```

## Результат
Лучшая модель - градиентный бустинг.

## Оценка качества
- MAE: 0.07
- RMSE: 0.085
- MAPE: 39.85
- Directional Accuracy: 0.54

## Интерпретация результатов
- Метрики MAE и RMSE находятся на уровне, который можно считать приемлемым для прикладного использования.
- Показатель directional accuracy демонстрирует, что модель достаточно уверенно отслеживает направление изменений target.
